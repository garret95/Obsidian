# 1. Vista General

**Alineamiento**: No alineado

**Género**: Hombre

**Raza**: Humano

**Clase**: Monje ascendido

**Edad**: Adulto

**Rol del Personaje**: Posible enemigo

**Condición**: No-muerto

# 2. Perfil

Va ataviado con una túnica de monje y una [[Máscara de Cieloduro]]. Destacan sus garras afiladas sin piel.

# 3. Historia

El primer no-muerto ascendido a sus órdenes que [[Arthur Larios]] consiguió obtener. Vigila desde hace siglos la biblioteca de [[Selid]]. Si observa malos comprtamientos para con el edifico o alguien intenta robar algún libro, se las verá con aquel que conserva la historia.

# 4. Vida Personal

No tenía contactos cercanos.

# 5. Vida Profesional

Ciudadano, actualmente conservador de la biblioteca y esbirro de [[Arthur Larios]].

# 6. Posesiones

Armadura, armas, inventario, dinero, vivienda, mascotas...

# 7. Habilidades

Patoso, rápido, débil, frágil, fuerte, ágil, pesado, sigiloso, resistente...

# 8. Recursos

Suertudo, popular, influente, anónimo, incorrompible, famoso, indefenso, irrelevante, aislado, de clase baja, pobre, poderoso, con reputación, inculto, impopular, de clase alta, rico, con contactos, con estudios...
