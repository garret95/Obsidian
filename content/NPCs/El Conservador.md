# 2. Perfil

Va ataviado con una túnica de monje y una [[Máscara de Cieloduro]]. Destacan sus garras afiladas sin piel.

# 3. Historia

El primer no-muerto ascendido a sus órdenes que [[Arthur Larios]] consiguió obtener. Vigila desde hace siglos la biblioteca de [[Selid]]. Si observa malos comprtamientos para con el edifico o alguien intenta robar algún libro, se las verá con aquel que conserva la historia.
