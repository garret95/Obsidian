Tienes una limitada capacidad de aguante que puedes usar para protegerte del peligro. 
- En tu turno, puedes usar una acción adicional para ganar una cantidad de puntos de golpe igual a 1d10 + tu nivel de guerrero.
- Una vez que hayas usado este rasgo, debes hacer un descanso corto o prolongado antes de poder volver a usarlo.
